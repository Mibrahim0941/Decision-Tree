# Decision Tree Implementation (CART)

This repository contains the work I completed as part of my internship. The primary focus of this project was to gain a deep, practical understanding of the Classification and Regression Trees (CART) algorithm from the ground up.

## Project Overview

To truly grasp the inner workings of decision trees, I approached the problem in two phases:

1. **Theoretical Understanding (Solving on Paper)**
   Before writing any code, I manually worked through the mathematics and logic of the CART algorithm on paper. This involved understanding how the algorithm evaluates features, calculates metrics (like Gini Impurity or Entropy), and decides the optimal splits to build the tree structure.

2. **Practical Implementation (Code & Titanic Dataset)**
   After building a solid theoretical foundation, I translated my manual calculations into a working code implementation. To test and demonstrate its effectiveness, I applied the custom-built CART algorithm to the well-known **Titanic dataset**, predicting passenger survival based on various features.

## Files in this Repository

- **`CART_dt.ipynb`**: The main Jupyter Notebook containing the code implementation of the CART algorithm and its application to the Titanic dataset.
